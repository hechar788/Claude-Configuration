# Feature Implementation Plans

This directory contains feature plan files generated by the `/plan-feature` command.

Each file follows the naming pattern: `feature-plan-<feature-name>.md`

These documents contain:

- Architecture decisions
- File changes (new and modified)
- Step-by-step implementation phases
- Component and hook specifications
- Data flow diagrams
- Edge case handling
- Testing checklists

Use these as implementation guides when building features.
